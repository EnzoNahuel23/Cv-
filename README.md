# CV Web - Carnero Enzo Nahuel

Este proyecto consiste en un Currículum Vitae técnico desarrollado utilizando HTML5 y CSS3, con una estructura semántica y un diseño simple, claro y profesional.

## Descripción

Se trata de una página web estática donde presento mi perfil profesional, mis tecnologías principales, un proyecto destacado y mis datos de contacto.

El objetivo del proyecto es demostrar conocimientos en:

- Maquetado semántico

- Separación de estructura y estilos

- Buenas prácticas básicas de desarrollo frontend

- Organización clara del contenido

## Tecnologías utilizadas

- HTML5 – Estructura semántica (header, main, section, article, footer)

- CSS3 – Estilos, diseño visual y maquetado básico

## Funcionalidades

- Presentación de perfil profesional

- Sección de tecnologías técnicas defendibles

- Sección de proyectos con enlace a GitHub

- Información de contacto

- Diseño simple y legible

- Código organizado y fácil de mantener

## Proyecto Destacado

Stock Store
Página web básica de venta de indumentaria urbana desarrollada como práctica de maquetado y organización de contenido.
